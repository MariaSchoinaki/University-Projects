# **Python Programming - Assignment 2**

## **Overview**
This assignment consists of **Python programming exercises** focusing on **recursion, functions, input handling, numerical computations, and functional programming concepts**. The problems require implementing various mathematical and string manipulation functions while adhering to structured programming principles.

---

## **Topics Covered**
### **1. Digit Sum Computation**
- Implementing a function to compute the **sum of digits** of a given number.

### **2. Counting Specific Digits**
- Implementing a function to count occurrences of a specific **digit at a given position** in a number.

### **3. Summing a Mathematical Series (π Approximation)**
- Implementing a function to compute the sum of an **alternating series** used for approximating π.

### **4. Generating a Star Pattern**
- Implementing a function that returns a string of **n asterisks ('*')**.

### **5. Function Argument Reversal**
- Implementing a function that **reverses the order of arguments** in another function.

### **6. Recursive Geometric Computations**
- Implementing **recursion-based functions** for computing geometric properties like **circle circumferences and square areas**.

### **7. Creating Quadratic Functions**
- Implementing a function that returns a **quadratic function generator**.

### **8. Abacus Representation of Operators**
- Implementing a function that converts **numeric operations** (addition, multiplication) into a visual **abacus-style representation**.

### **9. Recursive Sum of Squares**
- Implementing a function that computes the **sum of squares** from `1` to `n` using recursion.

### **10. Recursive Digit Printing**
- Implementing a function to **print the digits of a number recursively** in the correct order.
